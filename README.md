# 間取りプランナー (Madori Planner)

ブラウザだけで動く、間取り＋家具配置のプランニングアプリ。インストール不要で、`index.html` を開くだけで使えます。

## 主な機能

- **部屋/スペースの作成**：部屋・ベランダ・吹き抜け・WIC・階段・物置・お風呂・芝生・ポーチ・駐車場・駐輪場。いずれも長方形の組み合わせ（L字・凸型）で作れ、辺どうしが自動で吸着します。
- **家具・設備の配置**：ベッド/ソファ/テーブル/キッチン/カウンター類、建具（ドア・両開き・引き戸・窓）、電気（コンセント・エアコン・室外機）、AV（スクリーン・プロジェクター）、屋外（車4種・水栓）、汎用の箱。
- **編集**：クリック選択／Shift+クリックや範囲選択で複数選択／選択をドラッグでまとめて移動／回転（R）／複製／重なり順（最前面・最背面）／1cm刻みの移動。
- **計測ツール**：始点からマウス位置までの長さ（mm/m）を表示。
- **電卓**：長さ計算（+ − × ÷）と結果を幅/奥行へ反映。
- **取り消し/やり直し**：Ctrl+Z / Ctrl+Y。
- **保存**：localStorageへ自動保存、JSON書き出し/読み込み、PNG画像出力。
- グリッド1マス = 910mm（半間）。

## 使い方

`index.html` をブラウザで開くだけ。サーバー不要です。

## 構成

- `index.html` — 画面とUI
- `styles.css` — スタイル
- `app.js` — 描画・操作・保存などのロジック（Canvas / Vanilla JS）
