プロパティ

家具または部屋を選んでください

L字や凸型は「＋長方形を追加」。Alt(Option)+ドラッグでその長方形だけ移動して形を調整、サイズは下の幅/奥行で数値指定。ふつうのドラッグは部屋ごと移動します。

電卓 (長さ計算)

＝

+ − × ÷ ( ) が使えます。Enterで計算。結果は選択中の幅/奥行に「→幅」「→奥行」で入れられます。

操作: クリックで選択 / Shift+クリックで複数選択 / 空欄をドラッグで範囲選択 / 選択したものをドラッグで一緒に移動 / Delで削除 / Rで回転 / ホイールで拡大縮小 / 右ドラッグで画面移動。一緒に動かしたい物は複数選択してからドラッグ（上に乗っているだけの物は動きません）。ドア/窓は壁の上に置くと開口になります。
